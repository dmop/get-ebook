Ebooks microservice

    Only for the epubr.club site
    Search for the book you want on the /search route
    Pass the url of your book to return the download link on the /ebook route

Example usage:

https://get-ebook.herokuapp.com/search/cronicas de duna

Example output:

[ { "title": "Filhos de Duna – Crônicas de Duna #03 – Frank Herbert", "url": "http://epubr.club/filhos-de-duna-cronicas-de-duna-03-frank-herbert/" }, { "title": "Messias de Duna – Crônicas de Duna #02 – Frank Herbert", "url": "http://epubr.club/messias-de-duna-cronicas-de-duna-02-frank-herbert/" }, { "title": "Duna – Crônicas de Duna #01 – Frank Herbert", "url": "http://epubr.club/duna-cronicas-de-duna-01-frank-herbert/" }, { "title": "Cinza do tempo – Gustavo Barroso", "url": "http://epubr.club/cinza-do-tempo-gustavo-barroso/" } ]

Example usage:

https://get-ebook.herokuapp.com/ebook/http://epubr.club/duna-cronicas-de-duna-01-frank-herbert/

Example output:

[ { "type": " KINDLE", "url": "http://kimechanic.com/1Dky" }, { "type": " EPUB", "url": "http://kimechanic.com/1Dl9" } ]
