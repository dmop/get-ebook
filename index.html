Ebooks microservice

    Only for the epubr.club site
    Pass the url of your book to return the download link

Example usage:

https://get-ebook.herokuapp.com/http://epubr.club/duna-cronicas-de-duna-01-frank-herbert/

Example output:

[ { "type": " KINDLE", "url": "http://kimechanic.com/1Dky" }, { "type": " EPUB", "url": "http://kimechanic.com/1Dl9" } ]
